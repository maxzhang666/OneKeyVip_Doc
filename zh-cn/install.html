# 前言

如果你想顺利使用来自数码喵提供的插件，请务必阅读完成本篇教程。

# 安装Tampermonkey

# 方案一（360极速浏览器）

1.在360极速浏览器右上角，找到扩展商店的图标，并进入扩展商店。

1.在360极速浏览器扩展商店中，搜索“Tampermonkey”

2.在搜索结果中点击“安装”，即可完成“Tampermonkey”的安装。

# 方案二（360极速浏览器）

1.你可以在Chrome插件商店中进行在线安装，但这需要一定的技巧才能访问。

在线安装：在线安装 (opens new window)

2.如果你没有此类技巧，可以下载“Tampermonkey”的.crx离线包进行安装。

点击下载：离线插件包 (opens new window)

3.在Chrome浏览器中，点击右上角的“更多”按钮>“更多工具”按钮>“扩展程序”按钮，进入扩展程序管理页面

3.只需要在浏览器中打开“开发者模式”，再将“Tampermonkey.crx”的扩展名称由.crx更改为压缩包格式.zip，并解压成文件夹进行加载。

4.解压完成后，点击加载已解压的文件夹，即可顺利运行并使用。

5.但需要注意的是，开发者模式需要一直保持开启状态，否则将可能无法正常使用或“Tampermonkey”图标消失。
