# 📌使用教程

在使用脚本之前，请先确认你是否已安装Tampermonkey
如果没有安装Tampermonkey，脚本将无法正常提供服务

# ⌨️网盘提取码识别功能

展开获得详细教程 

1.鼠标选择百度网盘的链接及提取码，即可自动识别。






2.在识别后按照提示框进行打开
2.1.顺利打开后将自动填充提取码。

# 🔃网盘直链解析功能

展开获得详细教程 

1.将需要下载的文件（压缩包）保存至你的网盘内
1.1.并在网盘内选择要下载的文件
1.2然后点击名为网盘工具箱-直链的按钮进入解析页面






2.进入解析页面后
2.2.点击【配置卡密】填写在右侧公众号中获取的卡密
2.3.再点击解析按钮即可完成解析




3.打开IDM，并在下载>选项>下载>粘贴在脚本中复制的UA








4.完成以上设置后，点击【新建任务】填入直链





5.等待下载完成

# 📚Aria与Motrix的使用教程

展开获得详细教程 

1.在解析完成后，点击【脚本配置】进行配置

1.1.或在解析前点击【脚本配置】进行Aria的地址端口及下载保存目录进行配置




2.在【Aria配置】中将Aira RPC地址设置为:http://localhost:6800/jsonrpc

2.1.Aria RPC密钥可为空，默认的下载地址可进行自定义。




3.在配置完成后，打开下载好的Aira软件包，找到【AriaNg启动器.exe】并双击启动。

3.1.在启动完成后，会弹出AriaNg的web端页面，配置正确时左下方中部会出现【已连接】的提示。



4.返回到脚本页面，点击【发送到Aria】



4.1.一切配置正确时，在发送后脚本会提示发送成功。此时在AriaNg的web页面可以看见已开始下载。



5.如果你想采用Motrix进行下载时，只需要点击【脚本配置】按钮

5.1.在Aria配置中将Aria RPC地址的端口6800更改为16800即可。




5.2.在使用Motrix时需要注意的是，你的文件保存目录必须要和Aria配置中的相同。



如果你对于Aria的配置不是特别熟悉或是第一次使用

建议不要对Aria RPC地址及密钥进行修改

只需要对你的文件保存目录进行修改即可。
