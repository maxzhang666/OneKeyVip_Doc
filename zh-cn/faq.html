# 使用指南

下载提示 数码小站在线解析服务需要入群获取验证码，入群时请查看QQ等级是否≥9级及是否为退群黑名单，当你获取验证码之后进行退群将无法再次入群获取验证码使用本站。 因百度限制，需修改浏览器 User Agent后下载。你可以在下方选择你喜欢的方式进行下载。

# aria2（推荐）

[点击此处](http://pan.shuma.ink/resource/aria2.zip) 下载 aria2.zip
解压 aria2.zip 文件并运行其中的 点此启动.bat
打开解析下载页面，点击 推送到Aria2

# IDM

选项 -> 下载 -> 手动添加任务时使用的用户代理（UA）-> 填入shuma
右键复制下载链接（如果 直接点击 或 右键调用IDM 将传入浏览器的 UA 导致下载失败），在 IDM 新建任务，粘贴链接即可下载。

# ADM Pro（Android推荐）

设置 -> 下载中 -> 浏览器标识 -> 自定义 浏览器标识
填入 shuma
切换到浏览器（ADM留在后台），长按“下载链接”，选择复制链接地址
然后在ADM里点击开始即可

# Chrome 浏览器

安装浏览器扩展程序 [User-Agent Switcher for Chrome](https://chrome.google.com/webstore/detail/user-agent-switcher-for-c/djflhoibgkdhkhhcedjiklpkjnoahfmg)
右键点击扩展图标 -> 选项
New User-Agent name 填入 百度网盘分享下载
New User-Agent String 填入 shuma
Group 填入 百度网盘
Append? 选择 Replace
Indicator Flag 填入 Log，点击 Add 保存
保存后点击扩展图标，出现“百度网盘”，进入并选择“百度网盘分享下载”。
Chrome 应用商店打不开或者其他 Chromium 内核的浏览器，[请点此下载](http://pan.shuma.ink/resource/UserAgentSwitcher.crx)
[QQ浏览器插件下载](https://appcenter.browser.qq.com/search/detail?key=User-Agent%20Switcher%20for%20Chrome&id=djflhoibgkdhkhhcedjiklpkjnoahfmg%20&title=User-Agent%20Switcher%20for%20Chrome)

# Pure 浏览器（Android）

设置 –> 浏览设置 -> 浏览器标识(UA)
添加自定义 UA:shuma

# Alook 浏览器（IOS）

设置 -> 通用设置 -> 浏览器标识 -> 移动版浏览器标识 -> 自定义 ->
填入 shuma
