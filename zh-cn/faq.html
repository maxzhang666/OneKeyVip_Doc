# 网盘解析使用指南

# IDM

  • IDM下载:点击此处 (opens new window)
  • 选项 -> 下载 -> 手动添加任务时使用的用户代理（UA）-> 填入shuma
  • 右键复制下载链接（如果 直接点击 或 右键调用IDM 将传入浏览器的 UA 导致下载失败）
  • IDM 新建任务，粘贴链接即可下载。

# ADM Pro（Android推荐）

  • 设置 -> 下载中 -> 浏览器标识 -> 自定义 浏览器标识
  • 填入 shuma
  • 切换到浏览器（ADM留在后台），长按“下载链接”，选择复制链接地址
  • 然后在ADM里点击开始即可

# Chrome 浏览器

  • 安装浏览器扩展程序 User-Agent Switcher for Chrome (opens new window)
  • 右键点击扩展图标 -> 选项
  • New User-Agent name 填入 百度网盘分享下载
  • New User-Agent String 填入 shuma
  • Group 填入 百度网盘
  • Append? 选择 Replace
  • Indicator Flag 填入 Log，点击 Add 保存
  • 保存后点击扩展图标，出现“百度网盘”，进入并选择“百度网盘分享下载”。
  • Chrome 应用商店打不开或者其他 Chromium 内核的浏览器，请点此下载 (opens new window)
  • QQ浏览器插件下载 (opens new window)

# Pure 浏览器（Android）

  • 设置 –> 浏览设置 -> 浏览器标识(UA)
  • 添加自定义 UA:shuma

# Alook 浏览器（IOS）

  • 设置 -> 通用设置 -> 浏览器标识 -> 移动版浏览器标识 -> 自定义 ->
  • 填入 shuma
