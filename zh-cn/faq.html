# 脚本配套工具下载

# IDM

  • IDM下载:点击此处 (opens new window)
  • 选项 -> 下载 -> 手动添加任务时使用的用户代理（UA）-> UA请在脚本内进行复制
  • 右键复制下载链接（如果 直接点击 或 右键调用IDM 将传入浏览器的 UA 导致下载失败）
  • IDM 新建任务，粘贴链接即可下载。

# aria

  • Aria下载:点击此处 (opens new window)
  • 点击【脚本配置】前两项设置保持默认即可，如果你的电脑中不存在D盘，请设置至其他盘符或指定任意路径。
  • 完成设置后，点击【开始.bat】。
  • 在脚本中点击【发送到Aria】即可开始下载。
  • 在AriaNg web页面可以查看文件下载进度及其他操作。

# Motrix

  • Motrix下载:点击此处 (opens new window)
  • 设置 -> 进阶设置 -> 安全性: 模拟用户代理(UA) -> UA请在脚本内进行复制。
  • 使用【发送到Aria】功能时，不需要配置UA，只需要在【脚本配置】 -> 【Aria配置】中将http://localhost:6800/jsonrpc中的6800端口更改为16800端口即可。
