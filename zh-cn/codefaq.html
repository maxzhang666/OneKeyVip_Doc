# 常见问题

# 1.数码小站解析网页版验证码不正确

  • 请输入与近期相符合的验证码，不一定会每天更新，以QQ群内最新一条验证码为准即可。

# 2.小程序获取的卡密不对

  • 若获取数码小站脚本卡密，需要在公众号[小木盒]中回复[777]获得小程序入口，在小程序入口中选择【777】数码小站工具箱卡密领取，使用获得的卡密进行使用即可。
  • 若获取kinhdown网页版及客户端版本卡密，需要在需要在公众号[小木盒]中回复[888]获得小程序入口，在小程序入口中选择【888】Kinhdown卡密领取，使用获得的卡密进行使用即可。
  • 若不按照正常流程进行获取，或错用卡密，都将造成无法使用或是卡密错误的问题。

# 3.卡密已过期或无效

  • 当前小程序问题BUG比较多，但受影响的均为小范围影响，多数无影响。
  • 卡密会定期2~3天进行清理，卡密清理后重新在公众号内获取即可，其他无任何影响。

# 4.数码小站客户端打开文件夹失败

  • 若分享链接内存在多个文件夹，则无法正常打开，你可以尝试使用数码小站 (opens new window)网页版进行解析，或是通过油猴脚本点这里安装油猴脚本。

# 5.网页版是否支持批量解析？

  • 网页版并不支持批量解析。
