# 常见问题

# 1.为什么一直提示写入缓存

  • 当遇此情况时，可能是由于服务器网络拥堵或服务器正在遭受攻击，请刷新页面重新选择文件进行解析。

# 2.提示svip帐号被限速怎么办

  • 出现该类情况时，请等待作者进行后台维护即可，维护时间不会太长。

# 3.流量能用多久？是否会失效

  • 流量卡密会一直使用至流量额度为0时失效，当流量卡密仍有额度时则不会失效。

# 4.批量下载与IDM使用问题

  • 脚本暂不支持批量下载与文件夹下载。
  • 当IDM不可用时，请确认你已正确安装IDM及根据教程检查用户浏览器UA，在脚本中并不会直接唤起IDM。

# 5.解析时为什么会出现401、403.html的错误

  • 解析链接在新建任务粘贴到IDM时，出现401、403.html这种情况可能是因为你的资源存在问题、或是用户代理UA未正确设置，请自行检查相关设置项后再继续。
  • IDM下载过程中断提示输入admin帐号及密码时，是因为后台SVIP被系统限速而造成了该帐号下所解析的资源被限制，无法继续高速下载，遇此情况时请重新解析资源进行下载。
  • 上述两种情况，需要用户获取到解析链接后尽快地下载完成，否则可能下载任务会被中断而导致失败。

# 6.解析链接的有效期是多久

  • 解析出来的直链最长有效期为8小时，在8小时后直链自动失效且需重新解析。
