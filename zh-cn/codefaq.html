# 常见问题

# 1.脚本提示:任务排队中，请稍后再试[脚本答疑]

  • 当遇此情况时，可能是由于服务器网络拥堵或服务器正在遭受攻击，请刷新页面重新选择文件进行解析。

# 2.网页版提示PCS地址错误或地址错误的相关提示[网页版答疑]

  • 请检查你的连接是否正确，某些19年以前的链接可能会存在一些问题，无法进行正确解析。

# 3.在IDM中为什么会出现无权限下载/403等弹窗提示[通用答疑]

  • 请根据教程在IDM中正确设置UA，出现这种情况多数情况为没有正确设置UA。

# 4.发送至Aria2时发生未知错误，请重试！[脚本答疑]

# 4.1.Aria2下载器配置尚未完成,请先配置Aria2下载器信息[脚本答疑]

  • 你都没有配置正确Aria或没有下载Aria，当然会错误了。

# 5.加群后广告太多

  • 群内并不是广告，这些都是每天人工搜集的优惠券信息然后推送到群里的。
  • 这也是一种支持我们持续发展的方式，唯有支持我们才能持续提供免费且稳定的白嫖服务。
  • 当然，你如果觉得这是广告，觉得不能忍受或不适合你，你可以选择拒绝使用数码小站的解析服务。
